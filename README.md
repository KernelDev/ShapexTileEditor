# Honeycomb Image Application (Delphi 12)

**Версия:** 1.0.1  
**Дата обновления:** 29.04.2026  
**Статус:** Готов к использованию

## Описание

Приложение для работы с изображениями через фильтр в виде сот (ячеек).

## Возможности

1. **Загрузка фильтра** - черно-белое изображение, где:
   - Черный цвет пропускает ячейки
   - Белый цвет блокирует ячейки

2. **Создание сетки** - пространство заполняется ячейками (квадратами):
   - Настраиваемый размер ячеек
   - Ячейки, не видимые через фильтр, автоматически пропускаются

3. **Работа с ячейками**:
   - Объединение ячеек по строкам
   - Объединение ячеек по столбцам
   - Разделение ячеек на части
   - Перетаскивание ячеек мышью

4. **Работа с изображениями**:
   - Загрузка отдельного изображения
   - Массовая загрузка нескольких изображений для замощения
   - Вставка изображения в ячейку
   - Масштабирование изображения в ячейке
   - Смещение изображения внутри ячейки

5. **Сохранение проекта**:
   - Сохранение всех настроек ячеек и изображений
   - Загрузка сохраненного проекта

## Инструкция по использованию

1. Загрузите фильтр (черно-белое изображение)
2. Настройте размер ячеек
3. Загрузите изображение(я) для заполнения ячеек
4. Используйте кнопки для объединения/разделения ячеек
5. Перетаскивайте ячейки мышью
6. Настройте масштаб изображения в выбранной ячейке
7. Сохраните проект для последующего использования

## Файлы проекта

- `HoneycombApp.dpr` - главный файл проекта
- `MainUnit.pas` / `MainUnit.dfm` - основной модуль формы
- `CellUnit.pas` - модуль класса ячейки

## Сборка

Откройте проект в Delphi 12 Alexandria и скомпилируйте.
